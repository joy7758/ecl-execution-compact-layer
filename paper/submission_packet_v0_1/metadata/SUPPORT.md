# ECL Support

ECL v0.1 is a research-software artifact, not a production support service.

## How to Report Issues

Use the GitHub issue tracker:

```text
https://github.com/joy7758/ecl-execution-compact-layer/issues
```

Include:

- Python version
- operating system
- command run
- input fixture or minimized trace
- expected result
- observed result
- whether the failure affects validation, adapter mapping, replay, SDK usage, or documentation

## Supported Scope

- deterministic local replay
- schema validation
- included fixtures and examples
- OpenAI Agents SDK-style fixture mapping
- LangChain-style fixture mapping
- SDK and dependency API behavior

## Out of Scope

- production deployment support
- external API debugging
- MCP registry integration
- external runtime conformance certification
- DOI, JOSS, or arXiv submission support beyond local packet preparation
