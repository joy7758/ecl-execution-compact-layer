## Title

ECL v0.1: Deterministic Replay for Agent Runtime Traces

## Description

ECL v0.1 is a small deterministic execution intermediate representation for replayable agent runtime traces.

This demo shows:

- OpenAI-style and LangChain-style trace normalization;
- the four ECL surfaces: state, intent, action, and evidence;
- one-command local verification with `make demo`;
- deterministic replay artifacts and stable hashes;
- explicit mapping-loss boundaries.
